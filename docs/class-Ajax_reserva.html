Class Ajax_reserva

Pensado para el backend del sistema de reservas. Este controlador esta preparado para responder a peticiones AJAX. Cada funcion del modelo "be_reservas_model" tiene su equivalente con prefijo "ajax", la cual convierte todos los resultados de la BD en format JSON. Esto permite la directa manipulacion en el navegador con JS.
